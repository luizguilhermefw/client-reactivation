# Client Reactivation SaaS

A SaaS platform focused on automated customer reactivation and retention, enabling businesses to increase recurring revenue through smart and personalized campaigns.

---

## 📌 Overview

Many businesses lose customers simply because they don’t follow up after a purchase.

This platform solves that problem by:

- Automating customer follow-ups
- Scheduling reactivation campaigns
- Targeting customers based on behavior and time
- Increasing customer lifetime value

---

## 🧠 Core Idea

> Turn past customers into future revenue automatically.

---

## ⚙️ Tech Stack

### Backend
- NestJS
- Prisma ORM
- PostgreSQL (Neon)

### Architecture
- Modular structure (NestJS)
- Multi-tenant system (Company → Users → Customers)
- Scalable SaaS-ready backend
